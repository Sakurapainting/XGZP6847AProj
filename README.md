### 上手指南



###### 开发前的配置要求

1. C/C++
2. STM32CubeMX + Keil5 (HAL)
3. STM32F103C8T6
4. XGZP6847A 压力传感器
5. and circuit components...


###### **安装步骤**

Clone the repo

```sh
https://github.com/Sakurapainting/XGZP6847AProj
```

### 接线：

#### 压力传感器：
1. 不接
2. stm32 3.3v
3. stm32 gnd
4. stm32 3.3v
5. stm32 GPIO PA5
6. stm32 gnd

#### USB转串口：
GND: stm32 gnd
RX: stm32 GPIO PA9
TX: stm32 GPIO PA10
